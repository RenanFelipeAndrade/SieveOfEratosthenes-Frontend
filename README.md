# Sieve of Eratosthenes

## Requirements

- Nodejs 16 or higher
- NPM 8 or higher

## Get started

Clone the repository

```bash
git clone https://github.com/RenanFelipeAndrade/sieve-of-eratosthenes.git
```

Inside the directory, install all dependencies

```bash
npm install
```

Now, run the project

```bash
npm run dev
```
